# 📝 KẾ HOẠCH DỰ ÁN – EDUTRACK

## 1. 🎯 Mục tiêu dự án

- Xây dựng ứng dụng di động giúp sinh viên **quản lý chương trình đào tạo cá nhân**.
- Cho phép sinh viên:
  - Theo dõi các học phần đã hoàn thành và còn thiếu.
  - Quản lý số tín chỉ tích lũy.
  - Thống kê tiến độ học tập (% hoàn thành).
- Giao diện **thân thiện – trực quan – dễ sử dụng**.

---

## 2. 📦 Phạm vi dự án

### ✅ Chức năng chính (Sinh viên):

- Quản lý chương trình đào tạo (danh sách môn học, số tín chỉ).
- Cập nhật kết quả học tập (điểm, trạng thái hoàn thành).
- Thống kê tiến độ: số môn đã học, số môn còn lại, % hoàn thành.
- Gợi ý học phần cần đăng ký trong kỳ tiếp theo.

### ⚙️ Công nghệ sử dụng:

- **Frontend (Mobile UI):** 📱 Flutter / React Native
- **Backend:** 🌐 Firebase / Node.js (REST API)
- **Cơ sở dữ liệu:** 🐬 MySQL
- **Quản lý mã nguồn:** 🐙 GitHub
